# SetNonAssemblyFlags Method
Sets flags that are not assembly-specific.  
  
## Syntax  
  
```cpp  
HRESULT SetNonAssemblyFlags(  
    AssemblyFlags afFlags  
) PURE;  
```  
  
## Parameters  
 `afFlags`  
 ALink flags.  
  
## Return Value  
 Returns S_OK if the method succeeds.  
  
## Requirements  
 Requires alink.h
