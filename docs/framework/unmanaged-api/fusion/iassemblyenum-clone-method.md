# IAssemblyEnum::Clone Method
Creates a shallow copy of this [IAssemblyEnum](../../../../docs/framework/unmanaged-api/fusion/iassemblyenum-interface.md) object.  
  
## Syntax  
  
```cpp  
HRESULT Clone (  
    [out] IAssemblyEnum   **ppEnum  
);  
```  
  
## Parameters  
 `ppEnum`  
 [out] A pointer to the copy.
